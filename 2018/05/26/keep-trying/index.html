是夜 听雨 话

嘈杂像是缝纫机规律的震动，细听又像是电视的白噪声，眼前模糊的雪花被惊雷映亮

在昨天，哦！现在应该说是前天了。刷GitHub时，突发奇想想看看有哪些开源托管在“同性交友”网站上。

雨又大了起来

然后我就查呀查呀查，翻呀翻呀翻。JavaScript？Pass. C#? Pass. Python? Pass.
直到我看到了一个都很难再被提起的小游戏——2048。突生兴趣，我是不是可以把它用学校给的教材库来实现呢？
哪怕暂时阉割掉一些功能。顿时就是心血来潮，
用玄幻小说的文风，那便是：
敏锐的神觉有所感应，那一块看似普通的灰色兽骨中似乎涌动着特殊的力量，在呼唤着他。

雨淅淅沥沥起来，仿佛听到屋顶的积水顺着管道流动的声音
云层间还发出低沉的轰鸣，像是极强的低音单元，能让整片世界感受它的频率，多么幸福

从早上的大物到晚上的通核，尽管很多东西都用不到，但还是不停地查，又忘。不过也并不该说这些，貌似？
然后终于在昨天动工并做出了半成品，真的是很开心了，所以忍不住发了说说，手动扶额。（真·原始代码附于文末）

void NewNum(bool wantDraw)
{
int p=NULL;
int zero[16];//空白格
int i,j=0;
int index;
p=&num[0][0];
for (i=0;i<16;i++)
if ((p+i)==0)
{
zero[j]=i;//顺序记下空白格的序号
j++;
};
if (j!=0)
{
srand((unsigned)time( NULL ));//随机数初始化 （为了实现这一点我们需要time.h）
index=zero[rand() % j];
p+=index;
if (wantDraw) FreshMainRect();//先刷新画面再出新块
p=(rand()%2)?2:4;
FreshMainRect();//再刷新
}
}
